
# Example project for unexpected-react

This project is an example of using [unexpected-react](https://npmjs.org/package/unexpected-react)

To get started, simple clone the project, and run `npm install`

To run the tests, run `npm test`.  The command line for `mocha` is in the package.json file, under `scripts`.

The tests are located under `src/tests`.
